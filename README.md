# Edge-YOSO

This is an implementation of YOSO (You Only Segment Once) with MMDetection framework. 

The project is part of my undergraduate thesis. The primary goal is to modify and optimize the YOSO architecture for real-time panoptic segmentation on edge devices.


Original Code (Detectron2): [hujiecpp/YOSO](https://github.com/hujiecpp/YOSO).
